NomnomDB v1.4.0
NomnomDB is a database administration framework. Developed by Edutamamedia.com. 

Getting Started
You have to install xampp or other application that support web server, database server, and php.
You just need to create your database structure and let nomnomDB manage the data.

Built With
HTML - Famous markup language for web template design
CSS - Styling sheet to improve user interface
Jquery - Used to set an interactive user experience
PHP - Hypertext to handle the backend logic and database connection
SQL - A common database language
